# Regression Performance Metrics

## Linear Regression Model

The target variable was movie rating, and the predictor variables were genre indicator columns.

---

## Predictive Performance Measures

- Mean Absolute Error (MAE): 0.8135
- Mean Squared Error (MSE): 1.0443
- Root Mean Squared Error (RMSE): 1.0219
- R-squared (R²): 0.0391

---

## Visualizations

- Scatter Plot (`scatter_plot.png`)
- Residual Plot (`residual_plot.png`)

---

## Interpretation

Linear regression was used to predict movie ratings using genre features.

MAE, MSE, RMSE and R² were used to evaluate the predictive performance of the model.

The residual plot was used to assess prediction errors, while the scatter plot compares actual ratings with predicted ratings.

---

## Conclusion

Linear regression provides a baseline model for predicting movie ratings based on genre information.
